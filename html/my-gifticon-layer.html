캠페인 새로 만들기

  1. 캠페인 구분
    • 캠페인은 고객님이 원하시는 발송 형태에 따라 3가지 중 선택할 수 있습니다.
      대량발송 :
      기프티콘 shop에서 상품 구입 후 고객에게 즉시 발송할 수 있으며 기프티콘 상품을 일괄 발송하는 경우 선택하세요.
      연동발송 :
      고객의 이벤트 참여를 통해 기프티콘이 실시간 발송되는 형태를 원하시는 경우 선택하세요.
      연동 발송은 최소 2~3주전에 신청하시고 연동 API 규격에 따라 자체 개발을 완료해야만 사용이 가능합니다.
      광고주 상품 :
      기프티콘 입점 상품이 아닌 기업고객의 자체 상품 샘플링이나 홍보를 원하시는 경우 선택하세요.
  2. 캠페인 ID
    • 캠페인 ID는 신청 시 자동으로 발급되며 캠페인 이력 관리를 위한 기본 단위 입니다.
  3. 캠페인 개요
    • 진행하시는 캠페인에 대한 소개를 간략히 작성해 주세요. (예: 00서비스 신규 회원 모집을 위한 경품 이벤트)
  4. 캠페인 URL
    • 진행(예정)하시는 프로모션의 WEB 페이지 URL을 입력해 주세요.
  5. 캠페인 명
    • 주의 : 한글 20자 이내로 작성하실 수 있으며 입력하신 캠페인 명은 수정이 불가합니다.
  6. 캠페인 기간
    • 프로모션 진행 기간을 입력해 주세요.

상품발송하기 도움말

  1. 발송 그룹 명
    • 1개의 캠페인 내에 다수 회차 발송이 있을 경우 발송 차수를 구분하기 위한 정보로 입력합니다.
  2. 회신번호 등록
    • 고객 문의를 위한 회신 번호를 입력하며, 자주 사용하는 회신번호를 관리 할 수 있습니다. 유의) 030으로 시작하는 번호는 통신사 blocking 정책으로 발송이 되지 않습니다. 기타 030, 050, 070, 1588, 1599, 1600 등의 국번에 대해 고객이 스팸 넘버로 등록한 경우에는 수신이 불가능 합니다.
  3. 수신자 번호 등록
    • 직접 입력이 가능하며 엑셀 업로드 시 1회에 1천개 수신번호 등록이 가능합니다. (자주 사용하시는 수신번호는 주소록에 등록해 사용하시면 이용이 편리합니다.)
  4. 발송유형 선택
    기본형 :
    1개의 이미지(뒷장) 꾸미기가 가능하며 기프티콘 상품 이미지(앞장)와 롤링되는 형태로 발송 욉니다. 고객이 WAP 접속 후 이미지를 다운받는 방식으로 핸드폰 대기화면으로 저장됩니다. 유의) 스마트폰의 경우는 WAP접속이 불가능하며 자동으로 MMS 전환되어 발송됩니다.
    MMS 형 :
    1개의 이미지 꾸미기가 가능하며 MMS 형태로 발송 됩니다.
  5. SMS 문구 작성
    • 고객이 문자 보관함에 받게 되는 일반 문자의 문구로 사용 됩니다.
  6. 이미지 꾸미기
    • My꾸미기 이미지 메뉴에서 등록 후 승인된 이미지만 사용 가능 합니다.
  7. 무선 페이지 꾸미기
    • 무선 사이트 접속이 가능한 Callback URL SMS 수신 후 접속하는 첫 페이지를 꾸밀 수 있습니다. (사용 안함 선택 시 기프티콘이 배경화면에 즉시 다운로드 됩니다.)
  8. 발송 시간 선택
    즉시 발송 :
    결제 후 즉시 발송 됩니다.
    예약 발송 :
    발송 시간을 설정 할 수 있습니다.(최대 15일)
